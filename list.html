HTML Lists

An unordered list create data in dot list

An ordered list create data with a number.

Unordered HTML List

An unordered html list start with the ul tag. and each list start with the li tag.

Example :

  • coffee
  • milk
  • tea

Ordered HTML List

An ordered list start with ol tag. and each list start with li tag.

Example :

  1. milk
  2. tea
  3. coffee

HTML Description Lists

A description list define description of each term.

The dl tag defines the description list, dt tag defines the name. And dd tag describe each term.

coffee
- coffee is too hot and sweet
milk
- this milk is cold
