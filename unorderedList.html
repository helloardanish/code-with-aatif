HTML Unordered Lists

The unordered list will be marked with bullets(small black circle) by default.

Example :

  • coffee
  • milk
  • tea

Unordered HTML List - Choose List Item Marker :

The CSS list-style-type property is used to define the style of the list item maker.

You can use any of them from given below.

  • disc
  • circle
  • squre
  • none

Example :

this example is uses disc (default)

  • coffee
  • milk
  • tea

this example is uses circle

  • coffee
  • milk
  • tea

this example is uses square

  • coffee
  • milk
  • tea

this example is uses none

  • coffee
  • milk
  • tea

Nested HTML Lists

Nested list is uses for write list inside list.

Example :

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk
