HTML Ordered Lists

The HTML list start with ol tag. Each list item starts with the li tag.

An ordered list can be numrical or alphabetical.

By default item will be mark with number.

Ordered HTML List - The Type Attribute

The type attribute of the ol tag, defines the type of the list item marker:

You can use any types from given below.

type description
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

Numbers Example:

  1. coffee
  2. tea
  3. milk

Uppercase Letters Example:

  1. milk
  2. tea
  3. coffee

Lowercase Letters Example:

  1. coffee
  2. milk
  3. tea

Uppercase Roman Numbers Example:

  1. coffee
  2. milk
  3. tea

Lowercase Roman Numbers Example:

  1. coffee
  2. milk
  3. tea

Control List Counting

By default an ordered list will start counting from 1. if you will set start attribute to number 50 then list start from 50,51,52,etc.

Example :

  1. coffee
  2. milk
  3. tea
